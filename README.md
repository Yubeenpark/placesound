# placesound
------------
AR을 활용하여 공간을 다양한 모델로 꾸민다.
그 모델은 3d sounds 즉, 사용자가 모델과 떨어진 거리에 비례하여 소리가 증감한다.
이를 통해 공간을 시각적으로, 청각적으로 디자인할 수 있게 된다. 

# How to play this app
------------
fianl.apk를 다운 받고 실행시킨다.
